### Origin Statement 
The Quantan communication architecture and its associated modalities (Intuit Lens, Tashuan, and Canaria) were conceived and developed by **Thomas Douge** as an exploration of relational semantics and multimodal language design.
___

## Abstract
Quantan is an engineered semantic architecture designed to generate meaning through relational structure rather than grammatical inflection. Instead of beginning with vocabulary or grammatical categories, the system begins with a minimal semantic kernel and a central relational operator, **BETWEEN**, from which spatial, causal, comparative, and possessive relations emerge.

The architecture separates several functional layers: Morphemia (phonological and morphological structure), Quantan (semantic engine), and Intuit Lens (spoken language interface). Because the semantic layer is independent of any single communication channel, the system can also be expressed through additional modalities including the one-hand sign system Tashuan and the melodic signaling mode **Canaria**.

The result is a unified communication architecture in which meaning remains consistent across speech, gesture, and melody while remaining usable for everyday communication.
___

## Introduction
Quantan is a constructed semantic architecture designed to explore how meaning can emerge from relational structure rather than from traditional grammatical categories such as inflection, case, or agreement. The system is organized around a minimal set of semantic anchors and a central relational operator, **BETWEEN**, through which spatial, causal, comparative, and possessive relations can be represented within a unified framework.

The architecture was developed to investigate whether a communication system could be designed in which semantic relations are primary and grammatical complexity is minimized. In many natural languages, meaning is distributed across large vocabularies and extensive systems of inflection. Quantan instead attempts to construct meaning from a small set of structural principles that remain consistent across different modes of expression.

To organize these principles, the project is structured as a layered architecture. At the base lies **Morphemia**, which provides phonological and morphological constraints. Above this layer is **Quantan**, the semantic engine in which relational structures generate meaning. The spoken language **Intuit Lens** serves as the primary human interface, while **Tashuan** and **Canaria** extend the architecture into gesture and melodic communication.

## Design Principles

The Quantan communication architecture was developed according to a set of guiding principles intended to maximize clarity, structural coherence, and adaptability across communication modes.

### 1. Minimal Semantic Kernel
 Meaning is generated from a small set of conceptual anchors rather than from large inventories of grammatical categories. This reduces structural redundancy while preserving expressive power.
### 2. Relational Structure Over Inflection
 Semantic relations are represented through structural relationships rather than through inflectional morphology such as case, agreement, or conjugation.
### 3. Predicate-Centered Event Representation
 Events are resolved by predicates that define how participants relate to one another. Clauses typically place contextual elements before the predicate, which closes the event structure.
### 4. Salience-Based Discourse Tracking
 Participants are tracked through discourse salience roles (MC, SC, NPC) rather than through pronouns or agreement systems. This allows reference to follow narrative focus.
### 5. Structural Transparency
 The architecture is designed so that the same relational structures remain visible across different communication modes and registers.
### 6. Multimodal Compatibility
 Semantic structure is independent of communication channel, allowing the same meanings to be expressed through speech, gesture, or melodic signaling.
### 7. Efficiency for Everyday Communication
 While the architecture supports formal analysis, the Normal register is designed to allow efficient communication without requiring speakers to consciously manage the underlying structure.

## Predicate System

Events are expressed through predicates that define their own argument frames.

*Typical predicate relations include*:

LOCATE(entity, place)
HAVE(holder, object)
CAUSE(agent, event)
COMPARE(entity, standard)

Because predicates encode relational structure, the language does not require case marking or agreement systems.
___

## Intuit Lens (Spoken Language)

Intuit is the spoken interface of the Quantan architecture.

Sentence structure is typically predicate-final.

*Example*:

paba kaga kigi
tada pubu
tada kugu

*Meaning*:

A person went to a place, saw an object, and took it.

Participants are tracked through discourse salience rather than pronouns.

MC – Main Character
 SC – Secondary Character
 NPC – Non-participant

## Canaria (Melodic Communication Mode)

Canaria expresses Quantan meanings through melodic signaling.

Signals are structured through pitch motifs rather than phonemes.

The system organizes melodic signals through a phase-based grammar that allows sequences of tones to represent relational meanings.
