### Origin Statement 
The Quantan communication architecture and its associated modalities (Intuit Lens, Tashuan, and Canaria) were conceived and developed by **Thomas Douge** as an exploration of relational semantics and multimodal language design.
___

## Abstract
Quantan is an engineered semantic architecture designed to generate meaning through relational structure rather than grammatical inflection. Instead of beginning with vocabulary or grammatical categories, the system begins with a minimal semantic kernel and a central relational operator, **BETWEEN**, from which spatial, causal, comparative, and possessive relations emerge.

The architecture separates several functional layers: Morphemia (phonological and morphological structure), Quantan (semantic engine), and Intuit Lens (spoken language interface). Because the semantic layer is independent of any single communication channel, the system can also be expressed through additional modalities including the one-hand sign system Tashuan and the melodic signaling mode **Canaria**.

The result is a unified communication architecture in which meaning remains consistent across speech, gesture, and melody while remaining usable for everyday communication.
___

## Introduction
Quantan is a constructed semantic architecture designed to explore how meaning can emerge from relational structure rather than from traditional grammatical categories such as inflection, case, or agreement. The system is organized around a minimal set of semantic anchors and a central relational operator, **BETWEEN**, through which spatial, causal, comparative, and possessive relations can be represented within a unified framework.

The architecture was developed to investigate whether a communication system could be designed in which semantic relations are primary and grammatical complexity is minimized. In many natural languages, meaning is distributed across large vocabularies and extensive systems of inflection. Quantan instead attempts to construct meaning from a small set of structural principles that remain consistent across different modes of expression.

To organize these principles, the project is structured as a layered architecture. At the base lies **Morphemia**, which provides phonological and morphological constraints. Above this layer is **Quantan**, the semantic engine in which relational structures generate meaning. The spoken language **Intuit Lens** serves as the primary human interface, while **Tashuan** and **Canaria** extend the architecture into gesture and melodic communication.

## Design Principles

The Quantan communication architecture was developed according to a set of guiding principles intended to maximize clarity, structural coherence, and adaptability across communication modes.

### 1. Minimal Semantic Kernel
 Meaning is generated from a small set of conceptual anchors rather than from large inventories of grammatical categories. This reduces structural redundancy while preserving expressive power.
### 2. Relational Structure Over Inflection
 Semantic relations are represented through structural relationships rather than through inflectional morphology such as case, agreement, or conjugation.
### 3. Predicate-Centered Event Representation
 Events are resolved by predicates that define how participants relate to one another. Clauses typically place contextual elements before the predicate, which closes the event structure.
### 4. Salience-Based Discourse Tracking
 Participants are tracked through discourse salience roles (MC, SC, NPC) rather than through pronouns or agreement systems. This allows reference to follow narrative focus.
### 5. Structural Transparency
 The architecture is designed so that the same relational structures remain visible across different communication modes and registers.
### 6. Multimodal Compatibility
 Semantic structure is independent of communication channel, allowing the same meanings to be expressed through speech, gesture, or melodic signaling.
### 7. Efficiency for Everyday Communication
 While the architecture supports formal analysis, the Normal register is designed to allow efficient communication without requiring speakers to consciously manage the underlying structure.

# Relational Geometry of Attention
*A Conceptual Model Linking Embodied Interaction, Salience, and Interpretation*

**A geometric model of how attention flows from body, to interactions, to interpretation and abstraction.**   

This project proposes a **geometric model of attention and interaction** that traces how awareness moves from **embodied action in the body**, through **tools and effects in the world**, and into **interpretation and abstraction**. Using a small set of relational primitives—such as *point, vector, boundary, surface, and volume*—the model describes how perception, action, and meaning can be understood as part of a single relational structure. The framework serves as a conceptual foundation for the experimental language project **Quantan / Intuit**, whose design explores how such relational structures might be expressed directly in language.

## Terminology Note
The present model uses several concepts that overlap with established terminology in linguistics and cognitive science, particularly the notions of **frame** and **ground** used in frame semantics, perception theory, and related fields.

In this document these traditional terms are occasionally used for orientation. However, the Quantan / Intuit framework adopts alternative terminology that more closely reflects the operational structure of the model.

Specifically, the model replaces the traditional pair:

Frame
Ground

with the following terms:

Evaluation
Expectation.

### Evaluation
**Evaluation** refers to the domain or mode in which a relation is being assessed.
Examples include spatial evaluation, temporal evaluation, quantitative evaluation, logical evaluation, and causal evaluation.
Evaluation therefore determines **what kind of comparison or judgment is being made**.
In conventional terminology, this roughly corresponds to what is often called a frame.

### Expectation
**Expectation** refers to the baseline pattern or stabilized condition within an evaluation domain.
It represents the condition against which deviations are detected.
Examples include expected positions, expected sequences, expected quantities, or expected outcomes.
Expectation therefore functions similarly to what is traditionally called ground, but emphasizes the predictive and evaluative nature of the baseline.

### Surprise
When observed conditions deviate from expectation, the system registers **surprise**.
Surprise represents the moment at which deviation becomes sufficient to activate attention.
Within the model, surprise therefore plays a central role in the transition from passive perception to active attentional engagement.

## Terminological Rationale
The alternative terminology used in the Quantan / Intuit framework emphasizes that perception and cognition are fundamentally **evaluative processes**.
Rather than describing static structural contexts (frames) and passive backgrounds (grounds), the model treats perception as a dynamic sequence involving:
Evaluation
→ Expectation
→ Deviation
→ Surprise
→ Attention
→ Interpretation
This terminology reflects the model’s central concern with **attention, deviation detection, and interpretive activity**.

## Compatibility with Existing Literature
The use of alternative terminology should therefore be understood not as a rejection of existing theoretical concepts, but as a **reinterpretation of them within a relational and attentional framework**.
Readers familiar with frame semantics or figure–ground theory may find the concepts broadly analogous, though the present model emphasizes their **evaluative and dynamic character**.

## Foundations of the Model
The model begins from a small set of structural assumptions about how perception and meaning organize themselves. These assumptions are not intended as empirical claims about brain mechanisms; rather, they function as **conceptual primitives** that allow complex interactions to be described geometrically.
The progression below outlines the foundational sequence used in the model.

### 1. Gestalt
Perception does not begin with isolated elements but with **organized wholes**.
Humans typically encounter the world as **structured patterns**, not as collections of independent parts. The mind therefore tends to stabilize experience into **gestalts**—coherent configurations that serve as perceptual ground.
In this framework, Gestalt provides the **initial field of organization**.

### 2. Between
Once a field exists, meaning emerges from relations **between elements** rather than from the elements alone.
The fundamental relational condition is therefore:
ANCHOR — BETWEEN — ANCHOR
The concept of **Between** defines the relational field within which differences, positions, and interactions become meaningful.
Between is thus treated as the **primary generator of relations**.

### 3. True / False (Distinction)
Within the relational field, distinctions can be drawn.
 The most minimal distinction is the binary contrast:
True / False
This does not refer exclusively to logical truth, but to the **basic act of differentiation**—the ability to separate one condition from another.
True / False therefore represents the emergence of **contrast within the relational field**.

### 4. Unity (U) and Indeterminacy
Before distinctions are resolved, relations may exist in a state of **indeterminacy**.
In this framework:
U
represents an unresolved unity or undifferentiated relation.
Indeterminacy occurs when a structure exists but its **interpretation is not yet fully specified**.
This concept later appears in the model as the always present **IR (Indeterminate Remainder)**.

### 5. Differential
Once distinctions occur within a relational field, differences can accumulate.
A **differential** is any measurable or perceivable difference between relational states.
Examples include:
spatial displacement


contrast


motion


deviation from expectation


Differentials are what allow **salience and attention** to emerge within the system.

These structures eventually give rise to the **body interaction map, attention cascades, and interpretive ladders** described later in the document.

## Layer 2 — Relational Primitives
To describe how attention, interaction, and interpretation are structured, the model introduces a small set of **relational primitives**. These primitives are not objects or categories; they describe **forms of relation**.
The purpose of these primitives is to provide a **geometric vocabulary** capable of representing interactions between bodies, tools, environments, and meanings.

### Primitive Set
The model currently employs the following primitives:
POINT
VECTOR
INTERVAL
BOUNDARY
SURFACE
VOLUME
BRANCH
BETWEEN

These primitives represent **structural relations rather than physical entities**.

## Primitive Descriptions
### POINT
A **point** represents a localized position or event.
Examples:
fingertip contact


gaze fixation


impact point of a tool


In interaction chains, extremities and effect locations often behave as points.

### VECTOR
A **vector** represents directed movement or orientation.
Examples:
limb movement


gaze direction


tool motion


Vectors express **directional relations between points**.

### INTERVAL
An **interval** represents separation or extension across space or time.
Examples:
distance between two points


duration of a motion


the span of a movement


Intervals allow vectors and points to exist across measurable difference.

### BOUNDARY
A **boundary** represents a separation between regions.
Examples:
the edge of a surface


contrast between visual regions


categorical distinctions


Boundaries are critical for perception because they mark **changes in structure**.

### SURFACE
A **surface** represents the exterior interface of an object.
Examples:
the surface of a table


the visible face of a tool


the skin of the body


Surfaces are where many interactions occur.

### VOLUME
A **volume** represents spatial mass or containment.
Examples:
the body’s center of mass


the mass of an object


spatial occupancy


Volume is associated with **stability and balance**.

### BRANCH
A **branch** represents a point where structures divide or connect.
Examples:
joints in the body


branching structures in objects


decision points in interpretation


Branches allow relational structures to **expand or diverge**.

### BETWEEN
Between represents the **relational field connecting two anchors**.
The model treats this relation as fundamental:
ANCHOR — BETWEEN — ANCHOR
Meaning emerges through the structure of relations within this field.

### FOCUS
Focus represents the **selection of a particular location within a relational field**.
Focus determines where attention is directed.
It does not create structure, but **selects among available structures**.

### NIL
Nil represents the **absence of a relation or expected structure**.
Nil is important because deviations from expectation often appear as **missing or disrupted relations**.

## Function of the Primitive Set
Together, these primitives form a **minimal relational geometry** capable of describing:
bodily interaction


tool use


perceptual salience


structural deviation


interpretive organization


More complex phenomena can be described as **configurations of these primitives** rather than requiring additional conceptual categories.

# Layer 3 — Embodied Interaction
The relational primitives introduced in the previous section can be observed directly in the way humans interact with their bodies, tools, and environments.
Human action typically unfolds along a **continuous interaction chain** that links bodily structure to changes in the world.
A simplified representation of this chain is:
HEAD → SPINE → HIPS / DAN TIEN → LIMB → EXTREMITY → TOOL → EFFECT → OBJECT
This chain represents a sequence through which **orientation, motion, force, and interaction propagate from the body into the environment**.

## The Body Interaction Map
The model interprets each stage of this interaction chain using the relational primitives defined earlier.
HEAD        → FOCUS / BETWEEN (reference frame)
SPINE       → VECTOR (axis of orientation)
HIPS        → VOLUME (center of mass and balance)
LIMB        → VECTOR / INTERVAL (motion transmission)
EXTREMITY   → POINT / VECTOR (contact control)
TOOL        → VECTOR / INTERVAL extension
EFFECT      → POINT event across INTERVAL
OBJECT      → SURFACE / VOLUME / BOUNDARY / BRANCH
This mapping connects **relational geometry with embodied interaction**.

## Interaction Dynamics
During action, attention typically migrates outward along the interaction chain.
HEAD → SPINE → HIPS → LIMB → EXTREMITY → TOOL → EFFECT → OBJECT
The **outermost point where control affects the world** becomes the center of attention.
For example:
In hammering, attention rests at the **nail–wood contact point**.


In painting, attention rests at the **brush–canvas interface**.


In musical performance, attention rests at the **sound produced**, not the fingers producing it.


Thus awareness often appears to **extend beyond the body into the environment**.

## Tool Extension
Tools function as **extensions of the extremities**.
When a tool is used, the interaction chain expands:
EXTREMITY → TOOL → EFFECT → OBJECT
The tool becomes part of the control system, and attention shifts toward the **effect produced by the tool** rather than the body part holding it.

## Attention Migration
The model therefore distinguishes two general movements of attention.
### Action direction
During active interaction:
body → tool → effect → object
Attention moves **outward toward the point of impact or change**.
### Interpretive direction
When action stops and observation begins:
object → interpretation → abstraction
Attention may move **upward into explanation, narrative, symbolism, or conceptual analysis**.

## Salience and Deviation
Objects within an environment often fade into perceptual background through repeated exposure.
Attention is typically reactivated when a **deviation from the established ground** is detected.
Examples include:
unexpected motion


contrast or edge changes


unusual color or sound


distortion of expected form


These triggers correspond to the **X-Quotient salience features** introduced earlier.

## Embodied Interpretation
When interpreting objects, observers often reconstruct the **action chain that could have produced them**.
For example:
a craftsperson examining a chair may infer the joinery and tools used


a musician hearing a performance may imagine the physical gestures producing the sound


This process reflects **embodied simulation**, where perception and interpretation reconnect to the underlying interaction structures.

# Layer 4 — Salience and Attention
While the previous layer described the structural chain of interaction, this layer describes the mechanisms that determine **where attention activates within that chain**.
Human perception cannot attend to everything simultaneously. Attention therefore operates through a system of **background formation, deviation detection, and selective activation**.

## Ground Formation
Repeated exposure causes elements of the environment to become **perceptual ground**.
Objects that remain stable within a setting gradually require less attention and become part of the background of experience.
For example:
furniture in a familiar room


objects in a workshop


recurring sounds in an environment


These elements become **structural ground**, allowing attention to remain available for new or changing information.

## Habituation
Habituation is the process through which repeated stimuli lose their ability to trigger attention.
Even objects of high intrinsic interest—such as works of art or musical passages—may fade into background ground when encountered repeatedly without change.
Habituation therefore stabilizes perception by reducing unnecessary attentional activation.

## Deviation Detection
Attention is typically reactivated when a **deviation from established ground** occurs.
Deviation may appear as:
motion or displacement


contrast or edge change


unusual color or sound


distortion of expected form


violation of category expectations


When deviation exceeds a perceptual threshold, the object shifts from **background ground** into attentional foreground.

## The X-Quotient
Some forms of deviation are especially effective at triggering attention.
 The model refers to this set of high-priority triggers as the **X-Quotient**.
Typical X-Quotient features include:
motion
contrast
color change
size difference
sound
emotion
faces or gaze
spatial displacement
novelty
prototype distortion
These features amplify the likelihood that deviation will produce an attentional shift.

## Attention Economy
Attention is a **limited resource**.
Because multiple stimuli compete for attention, many potential triggers remain inactive if attentional capacity is already engaged elsewhere.
For example:
a musician may fail to notice musical details while navigating a crowded space


a craftsperson may overlook structural details when focused on another task


Attention economy therefore regulates **when potential salience triggers become active**.

## Mode Activation
Individuals develop specialized perceptual modes through repeated experience and practice.
A mode represents a domain-specific pattern of attention that lowers the activation threshold for certain types of deviation.
Examples include:
musicians noticing pitch or rhythm deviations
8

carpenters noticing joinery or grain direction


painters noticing brushwork or color balance


Modes sharpen perception by making domain-relevant features more likely to trigger attention.

## Salience Activation

# Layer 5 — Interpretation and Abstraction
Once attention has been activated and focused on a particular element of the interaction chain, perception may remain at the level of immediate observation, or it may move into progressively deeper forms of interpretation.
The model describes this progression as an **interpretive ladder**, in which attention moves from direct perception toward increasingly abstract forms of meaning.

## Embodied Simulation
The first level of interpretation often involves **embodied reconstruction**.
Observers frequently attempt to understand objects or events by implicitly reconstructing the actions that could have produced them.
For example:
a craftsperson examining a chair may imagine the cutting and joining processes used to build it


a musician hearing a passage may imagine the gestures required to perform it


a viewer observing a sculpture may mentally simulate the shaping actions involved


At this stage, interpretation remains closely tied to **bodily interaction and motor understanding**.

## Narrative Interpretation
Beyond embodied reconstruction, observers may begin to construct **narrative explanations**.
Narrative interpretation organizes events or objects within a temporal sequence.
Examples include:
imagining how an object was created


considering the circumstances surrounding an event


assigning intention or purpose to an action


Narrative interpretation therefore introduces **temporal and causal structure**.

## Symbolic Meaning
Objects may also acquire meaning through **symbolic association**.
In this stage, interpretation connects objects or events to shared cultural or conceptual systems.
Examples include:
recognizing an object as an antique or artifact


interpreting gestures or signs within social conventions


associating forms with cultural symbolism


Symbolic interpretation moves attention beyond the immediate situation into **shared systems of meaning**.

## Abstraction
At the highest level of interpretation, attention may move into **abstract reasoning**.
Here, the object or event is considered in terms of general principles or conceptual categories.
Examples include:
evaluating whether an object conforms to or deviates from a conceptual standard


analyzing the structure of a system


comparing instances within broader conceptual frameworks


Abstraction allows observers to move from specific observations to **generalized understanding**.

## The Interpretive Ladder
The interpretive process can therefore be represented as:
OBJECT
→ EMBODIED SIMULATION
* Layer 7
→ NARRATIVE
→ SYMBOL
→ ABSTRACTION
Each stage introduces additional layers of structure while retaining connection to the original perceptual event. — If the process is followed along it’s **default path**.

This organization does not follow a single rigid pattern.
Instead, cognition uses **multiple architectural strategies simultaneously**, forming what Gary Marcus describes as a **kluge system**: a collection of workable solutions that evolved for **practical efficiency** rather than perfect design.
Three major organizational modes appear repeatedly:

homoarchical **systems** detect **pattern convergence**

# Layer 8 — Limitations and Future Work
The framework presented here is a **conceptual geometric model** intended to clarify structural relationships between embodied interaction, perceptual salience, and interpretive cognition. It should therefore be understood as an **exploratory framework rather than a completed theory**.
Several important limitations should be noted.

## Conceptual Scope
The model is not intended as a detailed account of neurological mechanisms, psychological processes, or perceptual physiology. Instead, it describes **structural relationships between elements of experience** at a conceptual level.
Empirical validation and neurological correspondence remain outside the scope of the present work.

## Simplification
The framework intentionally reduces complex phenomena to a **small set of relational primitives**. While this simplification helps reveal structural patterns, it necessarily omits many nuances present in real cognitive and perceptual systems.
Future refinements may introduce additional structure or clarify how existing primitives combine in more complex configurations.

## Domain Generality
The model attempts to unify observations across several domains, including:
bodily interaction


tool use


perceptual salience


interpretive cognition


Although these domains share structural similarities, the present framework does not claim that the same mechanisms operate identically across all contexts. The model should therefore be understood as an **organizing analogy rather than a universal explanatory law**.

## Mathematical Formalization
The current presentation uses geometric language to describe relational structure. However, the model has not yet been fully formalized mathematically.
Possible future work may explore connections to:
graph theory


topology


dynamical systems


information theory


symmetry and entropy models


Such investigations may help clarify the deeper mathematical structure of the framework.

## Empirical Investigation
Several aspects of the model invite empirical investigation, including:
the role of deviation detection in attention activation


the influence of domain-specific perceptual modes


the relationship between embodied interaction and interpretive reasoning


the salience effects described by the X-Quotient triggers


Future research may explore whether these proposed structures correspond to measurable cognitive or perceptual patterns.

## Development of the Quantan / Intuit Language
The present model also serves as the conceptual foundation for the experimental language project **Quantan / Intuit**. Ongoing work in that project explores how relational primitives, interaction chains, and attention structures might be represented within a coherent linguistic system.
Further development of the language may provide additional insights into the strengths and limitations of the underlying conceptual framework.

## Open Framework
This model should therefore be regarded as an **open conceptual framework**. Its primary purpose is to organize observations about attention, interaction, and interpretation in a way that invites further exploration, critique, and refinement.
